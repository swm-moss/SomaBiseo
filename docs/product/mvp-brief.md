# SomaBiseo MVP Brief

SomaBiseo는 소프트웨어마에스트로 연수생이 공지, 특강, 멘토링 일정을 한곳에서 확인하고 자신의 Google Calendar와 충돌 여부를 확인한 뒤 필요한 일정을 캘린더에 저장할 수 있게 해주는 모바일 퍼스트 비공식 비서 서비스입니다.

## MVP 1차 포함

- 로그인 mock
- 대시보드
- 공지 목록과 상세
- 멘토특강, 자유멘토링 목록과 상세
- 관심 일정 저장
- Google Calendar 연결 mock
- 일정 충돌 확인 mock
- 캘린더 추가 mock

## MVP 1차 제외

- SOMA 특강 신청
- SOMA 특강 취소
- 자유멘토링 신청
- 자동 신청과 자동 취소
- 후기 커뮤니티
- 랭킹과 추천 알고리즘

## 핵심 승부처

1. 공지를 놓치지 않게 합니다.
2. 특강과 멘토링을 한눈에 보여 줍니다.
3. 내 캘린더와 겹치는지 바로 알려 줍니다.
